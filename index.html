§01

La Circonferenza Goniometrica

DEFINIZIONE
Circonferenza goniometrica

Circonferenza di centro $O=(0,0)$ e raggio $1$, equazione $x^2+y^2=1$. Ad ogni angolo $\alpha$ si associa il punto $P(\alpha)=(\cos\alpha,\sin\alpha)$ ruotando $(1,0)$ di $\alpha$ radianti in senso antiorario.

DEFINIZIONE
Seno, coseno, tangente

$\cos\alpha$ = coordinata $x$ di $P$,   $\sin\alpha$ = coordinata $y$ di $P$,   $\tan\alpha = \sin\alpha/\cos\alpha$ (definita per $\cos\alpha\neq0$).

IDENTITÀ FONDAMENTALE
$$\sin^2\alpha + \cos^2\alpha = 1 \qquad \forall\,\alpha\in\mathbb{R}$$
QuadranteGradi$\sin\alpha$$\cos\alpha$$\tan\alpha$
I$0^\circ\!<\!\alpha\!<\!90^\circ$+++
II$90^\circ\!<\!\alpha\!<\!180^\circ$+−−
III$180^\circ\!<\!\alpha\!<\!270^\circ$−−+
IV$270^\circ\!<\!\alpha\!<\!360^\circ$−+−
PROPRIETÀ
Periodicità

$\sin$ e $\cos$ hanno periodo $2\pi$. La $\tan$ ha periodo $\pi$.

§01 — Circonferenza goniometrica: punto a 60°
ES 01.1 — Valori notevoli

Calcola $\sin(135^\circ)$, $\cos(210^\circ)$ e $\tan(315^\circ)$.

// 135° = 180° − 45° → II quadrante → sin positivo
$$\sin(135^\circ) = \sin(45^\circ) = \frac{\sqrt{2}}{2}$$
// 210° = 180° + 30° → III quadrante → cos negativo
$$\cos(210^\circ) = -\cos(30^\circ) = -\frac{\sqrt{3}}{2}$$
// 315° = 360° − 45° → IV quadrante → tan negativo
$$\tan(315^\circ) = -\tan(45^\circ) = -1$$
ES 01.2 — Identità fondamentale

Dato $\sin\alpha=3/5$, $\alpha$ nel II quadrante. Trova $\cos\alpha$ e $\tan\alpha$.

// identità fondamentale
$$\cos^2\alpha = 1 - \frac{9}{25} = \frac{16}{25}$$
// II quadrante → cos negativo
$$\cos\alpha = -\frac{4}{5}$$
// tan = sin / cos
$$\tan\alpha = \frac{3/5}{-4/5} = -\frac{3}{4}$$

§02

Funzioni Trigonometriche

DEFINIZIONE
Funzione seno

$\sin:\mathbb{R}\to[-1,1]$. Dispari, periodo $2\pi$. Zeri: $x=k\pi$.

DEFINIZIONE
Funzione coseno

$\cos:\mathbb{R}\to[-1,1]$. Pari, periodo $2\pi$. Zeri: $x=\pi/2+k\pi$.

DEFINIZIONE
Funzione tangente

$\tan:\mathbb{R}\setminus\{\pi/2+k\pi\}\to\mathbb{R}$. Dispari, periodo $\pi$, strettamente crescente in ogni intervallo del dominio.

§02 — sin x (blu), cos x (verde), tan x (rosso) su [−2π, 2π]
ES 02.1 — Segno del seno

Dove è $\sin x>0$ e dove $\sin x<0$ in $[-2\pi,2\pi]$?

// sin x = 0 in x = kπ
$$\sin x > 0 \;\iff\; x\in(-2\pi,-\pi)\cup(0,\pi)$$ $$\sin x < 0 \;\iff\; x\in(-\pi,0)\cup(\pi,2\pi)$$
ES 02.2 — Simmetria complementare

Dimostra che $\cos(\pi/2-x)=\sin x$.

// formula di addizione
$$\cos\!\left(\tfrac{\pi}{2}-x\right)=\cos\tfrac{\pi}{2}\cos x+\sin\tfrac{\pi}{2}\sin x = 0\cdot\cos x+1\cdot\sin x=\sin x\quad\square$$

Equazioni Trigonometriche

TEOREMA
$\sin x = a$  (|a| ≤ 1)

$x = \arcsin a + 2k\pi$   oppure   $x = \pi - \arcsin a + 2k\pi$

TEOREMA
$\cos x = a$  (|a| ≤ 1)

$x = \pm\arccos a + 2k\pi$

TEOREMA
$\tan x = a$

$x = \arctan a + k\pi$

§03 — Soluzione di cos x = −½ in [0, 2π]: x = 2π/3 e 4π/3
ES 03.1 — cos x = −1/2

Risolvere: $\cos x = -\dfrac{1}{2}$

// arccos(−1/2) = 2π/3 (II quadrante)
$$x = \pm\frac{2\pi}{3} + 2k\pi, \quad k\in\mathbb{Z}$$
ES 03.2 — tan x = −1

Risolvere: $\tan x = -1$

// arctan(−1) = −π/4 → periodo π
$$x = -\frac{\pi}{4} + k\pi, \quad k\in\mathbb{Z}$$

§04 — Soluzione di sin x ≥ 0: zona verde = soluzione
ES 04.1 — cos x ≤ 0

Risolvere: $\cos x \leq 0$

// cos x = 0 in x = π/2 + kπ  |  cos ≤ 0 nel II e III quadrante
$$x\in\left[\frac{\pi}{2}+2k\pi,\;\frac{3\pi}{2}+2k\pi\right], \quad k\in\mathbb{Z}$$
ES 04.2 — tan x ≥ √3

Risolvere: $\tan x \geq \sqrt{3}$

// tan(π/3) = √3 → equazione associata: x = π/3 + kπ
// tan crescente → soddisfatta a destra di π/3, fino all'asintoto
$$x\in\left[\frac{\pi}{3}+k\pi,\;\frac{\pi}{2}+k\pi\right), \quad k\in\mathbb{Z}$$

§05

Identità e Angoli Doppi

TEOREMA
Formule di addizione
$$\sin(\alpha\pm\beta)=\sin\alpha\cos\beta\pm\cos\alpha\sin\beta$$ $$\cos(\alpha\pm\beta)=\cos\alpha\cos\beta\mp\sin\alpha\sin\beta$$
ANGOLO DOPPIO
$$\sin(2x)=2\sin x\cos x$$ $$\cos(2x)=\cos^2x-\sin^2x=2\cos^2x-1=1-2\sin^2x$$
PROPRIETÀ
Periodicità modificata

$\sin(nx)$ e $\cos(nx)$ hanno periodo $2\pi/n$.

§05 — sin x (blu, T=2π) vs sin(2x) (rosso, T=π): periodo dimezzato
ES 05.1 — 2cos²x − 1 ≥ 0

Risolvere: $2\cos^2x - 1 \geq 0$

// 2cos²x − 1 = cos(2x)
$$\cos(2x)\geq0$$
// t = 2x → cos t ≥ 0 → t ∈ [−π/2+2kπ, π/2+2kπ] → dividi per 2
$$x\in\left[-\frac{\pi}{4}+k\pi,\;\frac{\pi}{4}+k\pi\right], \quad k\in\mathbb{Z}$$
ES 05.2 — sin(2x) > 0

Risolvere: $\sin(2x) > 0$

// t = 2x → sin t > 0 → t ∈ (2kπ, π+2kπ) → dividi per 2
$$x\in\left(k\pi,\;\frac{\pi}{2}+k\pi\right), \quad k\in\mathbb{Z}$$

§06

Esercizi Graduati

sin x > −1/2

Risolvere $\sin x > -\dfrac{1}{2}$

// sin x = −1/2 → x = 7π/6 (III q.) e x = 11π/6 (IV q.)
// sin ≤ −1/2 in [7π/6, 11π/6] → soluzione = complementare
$$x\in\left(-\frac{\pi}{6}+2k\pi,\;\frac{7\pi}{6}+2k\pi\right), \quad k\in\mathbb{Z}$$
sin x · cos x ≥ 0

Risolvere $\sin x \cdot \cos x \geq 0$

// sin x cos x = ½ sin(2x) → sin(2x) ≥ 0
// t = 2x → t ∈ [2kπ, π+2kπ] → dividi per 2
$$x\in\left[k\pi,\;\frac{\pi}{2}+k\pi\right], \quad k\in\mathbb{Z}$$
tan x < 0

Risolvere $\tan x < 0$

// tan negativa nei quadranti II e IV (sin e cos di segno opposto)
$$x\in\left(-\frac{\pi}{2}+k\pi,\;k\pi\right), \quad k\in\mathbb{Z}$$
sin x − cos x ≥ 0

Risolvere $\sin x - \cos x \geq 0$

// forma armonica: R = √2, φ = π/4
$$\sin x - \cos x = \sqrt{2}\,\sin\!\left(x-\frac{\pi}{4}\right)\geq0$$
// t = x − π/4 → sin t ≥ 0 → t ∈ [2kπ, π+2kπ] → aggiungi π/4
$$x\in\left[\frac{\pi}{4}+2k\pi,\;\frac{5\pi}{4}+2k\pi\right], \quad k\in\mathbb{Z}$$
sin²x ≤ 1/4

Risolvere $\sin^2 x \leq \dfrac{1}{4}$

// sin²x ≤ (½)² ↔ |sin x| ≤ ½ ↔ −½ ≤ sin x ≤ ½
$$x\in\left[-\frac{\pi}{6}+k\pi,\;\frac{\pi}{6}+k\pi\right], \quad k\in\mathbb{Z}$$
sin x (2cos x − 1) ≥ 0

Risolvere $\sin x\,(2\cos x - 1) \geq 0$

// zeri: sin x = 0 → x = kπ  |  2cos x−1=0 → x = ±π/3+2kπ
